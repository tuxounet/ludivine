# USAGE

## via npm

```console
mkdir -p $HOME/.ludivine
cd $HOME/.ludivine
npm init -y
echo "@tuxounet:registry=https://npm.pkg.github.com" > .npmrc
npm install --save @tuxounet/ludivine

```
